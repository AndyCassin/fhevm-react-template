===============================================================================
                      DEMO VIDEO PLACEHOLDER
===============================================================================

The demo.mp4 file should be placed in this directory to showcase the FHEVM SDK
and example applications.

VIDEO SPECIFICATIONS:
-------------------
- Format: MP4 (H.264 codec)
- Resolution: 1920x1080 (minimum)
- Duration: 3-5 minutes
- Frame Rate: 30 fps (minimum)
- Audio: Clear voice narration
- File Size: Under 100MB

CONTENT TO INCLUDE:
-----------------
1. Introduction to FHEVM SDK
2. Next.js example demonstration
3. Patent License platform walkthrough
4. Code snippets and API usage
5. Key features highlight
6. Conclusion with call-to-action

For detailed instructions on creating the demo video, please refer to:
DEMO_VIDEO.md

ALTERNATIVE OPTIONS:
------------------
If the video file is too large for GitHub, consider:
1. Upload to YouTube and add link in README
2. Host on Vimeo
3. Use cloud storage (Google Drive, Dropbox) with public link

To add the video link to README, update the README.md file with:
[Watch Demo Video](https://youtu.be/your-video-id)
